Antes de começar, certifique-se de extrair todos os arquivos do ZIP para uma pasta no computador. Depois disso, é só executar o painel (painel.py). Esse arquivo foi projetado para ser o ponto de acesso principal às atividades, então você não precisará navegar pelos arquivos manualmente. O painel vai cuidar de tudo, garantindo que as atividades estejam disponíveis de forma organizada e fácil de executar.
